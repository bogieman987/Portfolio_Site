Math Sheet Calculator - Github

A typical Android calculator app, the goal of creating this was to learn how to utilize the Android SDK in making an app, as well as learning various other aspects to app design such as creating a good UI with a good UX and a set of decent features

Main features:

  • Basic calculator functions (addition, subtraction, multiplication, division)
    Status: complete

  • Basic memory functions (store in memory, add value to stored memory value etc.)
    Status: not started

  • Scientific calculator functions (brackets, log, exponents etc.)
    Status: Started work, initial bracket support completed

  • Plotting sum to grid (may add support for graphs)
    Status: not started

  • Saving/Loading previous sums with ability to name (potential addition of being able to store online via free cloud storage options)
    Status: not started
